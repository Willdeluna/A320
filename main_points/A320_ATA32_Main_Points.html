Core Concept

ATA 32 = motion, energy, and control→It supports the aircraft, moves the gear, and controls landing/taxi energy.big idea
Computer-controlled hydraulics→Logic commands hydraulic power, then sensors confirm mechanical movement.
Every sequence is sensor-confirmed→No valid sensor feedback can stop the next step from happening.why gear sticks
🔁

Landing Gear Sequence

Retract sequence→Lever UP → doors open → uplocks release → gear retracts → uplocks engage → doors close.
Extend sequence→Lever DOWN → doors open → gear extends → downlock engages → doors close.
Each step must be verified→Proximity sensors confirm position before logic allows the next action.sequence logic
🧠

LGCIU — Gear Brain

LGCIU controls sequence→It controls gear/door logic, monitors sensors, and sends ECAM data.not hydraulic
Only one LGCIU active→The active LGCIU alternates each cycle and can switch on failure.
Inputs and outputs→Inputs include gear position, door position, and WOW. Outputs include selector valve commands and indications.
📡

Proximity Sensors

Critical feedback source→Sensors report doors open/closed, gear up/down locked, and shock absorber/WOW state.
Most common failure source→False indications can block sequencing and trigger ECAM faults.common fault
Signal path matters→Proximity sensor → LGCIU → logic decision → ECAM/CFDS result.
💧

Hydraulic Control

Green system powers gear operation→Hydraulic power moves the gear, doors, locks, and actuators.
Electrical control directs hydraulic flow→LGCIU logic commands selector valves. Hydraulics do the muscle work.logic + power
Safety valve protects at high speed→The safety valve closes above high-speed limits to prevent unsafe operation.
🔒

Downlock / Uplock Logic

Hydraulics move it, mechanics hold it→Gear locks are mechanical holding devices, not just hydraulic pressure.huge
Downlock uses overcenter geometry→Springs and mechanical geometry help keep gear locked down.
Uplock uses hook/latch logic→Gear is mechanically held up after retraction.
Locks can be released hydraulically or mechanically→Free-fall extension mechanically releases uplocks when needed.
🪂

Free-Fall Extension

Emergency backup extension→Hydraulic pressure is removed, uplocks release, and gear falls by gravity/airflow.backup
Cut-out valve isolates pressure→Pressure is isolated and vent valves dump fluid to return.
Doors stay open→Free-fall gets the gear down, but normal door closing logic is not completed.
🛢️

Shock Absorbers

Oleo-pneumatic struts→Nitrogen acts as the spring and hydraulic fluid acts as the damper.
Absorbs landing energy→Shock struts reduce bounce and protect structure during landing and taxi.
WOW feedback is important→Shock absorber compression affects ground/air logic and system interlocks.WOW logic
🛞

Braking Modes

Normal braking→Green hydraulic system with BSCU control.
Alternate with anti-skid→Yellow system with BSCU/ABCU backup logic.
Alternate without anti-skid→Yellow system plus accumulator support.
Parking/emergency braking→Accumulator is the last-resort pressure source.backup layers
🧠

BSCU — Braking Brain

Controls braking and steering→BSCU manages braking pressure, anti-skid, autobrake, and nose wheel steering.brake brain
Redundant channels→Control and monitoring channels help prevent single-point failure.
Digital control, hydraulic power→Computers command the valves; hydraulic pressure does the stopping.
⚠️

Anti-Skid

Dynamic pressure modulation→Anti-skid compares wheel speed against aircraft speed and releases pressure if skid starts.very important
Prevents skids and tire damage→It protects tires and helps keep directional control during braking.
Wheel-speed data matters→Bad wheel speed inputs can create brake/anti-skid faults.
🛬

Autobrake

Autobrake controls deceleration→It is not just a fixed pressure setting. It targets aircraft deceleration.decel control
Modes include MAX, MED, and LO→MAX is for rejected takeoff. MED/LO are landing modes.
Ground spoiler deployment matters→Autobrake activation logic is tied to touchdown and spoiler deployment conditions.
Uses BSCU and ADIRU data→Autobrake depends on aircraft speed/deceleration data and brake control logic.
🅿️

Parking / Emergency Brake

Yellow system or accumulator→Parking and emergency brake pressure comes from the yellow hydraulic side or stored accumulator pressure.
Overrides normal braking→Parking brake logic can override other brake control paths.emergency brake
Accumulator gives limited applications→Stored pressure is limited, so brake applications are not unlimited.
↔️

Nose Wheel Steering

BSCU controls steering→Hydraulic actuators move the nose wheel based on BSCU commands.
Handwheel = large angle→Used for tight taxi steering and large nose wheel angles.
Pedals = small angle→Used for runway directional control with limited steering angle.
Speed-dependent authority→Steering authority reduces/disables as speed increases.speed limit
🛡️

Steering Protections

Feedback sensors protect control→Steering feedback helps prevent runaway steering and mismatch faults.
Anti-shimmy protection→Accumulator and damping logic help prevent wheel oscillation.
Overpressure protection→Valves protect components from excessive steering pressure.
📊

ECAM WHEEL Page

Primary diagnostic page→Shows gear position, door status, brake status, steering, and brake temperatures.start here
Color logic matters→Green is normal, red is unsafe, amber is fault/abnormal.
Use with CFDS→ECAM shows current state. CFDS helps identify stored LGCIU/BSCU history.
🔧

Common Failures & Line Reality

Most frequent faults→Proximity sensors, LGCIU logic faults, hydraulic leaks, valve issues, and brake temperature sensor faults.
Mechanical failure is less common→Many write-ups are indications, sensors, or logic problems before true structural failure.
Golden rule→Data first → hydraulics second → structure last.line MX mindset

🔧 Pro Troubleshooting Flow

1. ECAM WHEELCheck gear, doors, brakes, steering, and temps.
2. Identify AreaGear, brake, steering, or indication problem.
3. CFDSCheck LGCIU/BSCU last leg faults.
4. SensorsCheck proximity signals and WOW logic.
5. HydraulicsVerify Green and Yellow pressure paths.
6. MechanicalInspect locks, actuators, brakes, and structure.
